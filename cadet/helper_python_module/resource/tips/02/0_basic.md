# Garden Guardian - Exception Handling

## 이 모듈의 핵심

데이터는 항상 정상적이지 않습니다.

- 잘못된 숫자 입력
- 존재하지 않는 파일
- 0으로 나누기
- 타입이 다른 데이터 연산

등 다양한 문제가 발생할 수 있습니다.

Python의 Exception Handling은 프로그램이 비정상 종료(crash)되지 않도록 보호하는 핵심 문법입니다.

이번 모듈에서는:

- try / except
- raise
- finally
- 사용자 정의 Exception
- 여러 타입의 에러 처리

를 학습합니다.

목표는 "에러가 발생해도 계속 동작하는 프로그램"을 만드는 것입니다.
