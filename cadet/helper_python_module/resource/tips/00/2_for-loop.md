# for & range()

파이썬의 `for` 루프는 전통적인 카운터 기반 루프가 아닌, **이터레이터 항목을 순회하는 방식**입니다.

## 기본 문법

``` python
for "변수" in "이터러블"(Iterable):
    # 실행할 코드 block
```

이터러블에는 range() 외에도 데이터를 순차적으로 꺼낼 수 있는 다양한 타입의 객체가 들어올 수 있습니다.

## 대상 별 작동 원리

1. range()에 숫자가 들어갈 때 range(stop) 또는 range(start, stop[, step]) 형태로 사용됩니다. 내부적으로 숫자를 모두 메모리에 올리지 않고, 필요할 때마다 다음 숫자를 생성하는 이터러블 객체를 반환합니다.

2. 리스트/배열(List/Array)이 들어갈 때 배열의 첫 번째 요소(index 0)부터 마지막 요소까지 순서대로 변수에 바인딩하며 순회합니다.

3. 문자열(String)이 들어갈 때 문자열을 '문자(Character)들의 시퀀스'로 취급합니다. 가장 왼쪽 문자부터 한 글자씩 분리하여 순회합니다.

for ... in ... 문 뒤에 오는 대상은 반드시 **반복 가능한 객체(Iterable)** 여야 합니다. 단일 숫자(예: int 타입 5)는 이터러블이 아니므로 에러가 발생합니다.
